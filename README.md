# AI Berkshire · 个人投顾仪表盘

[![Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://ai-berkshire.streamlit.app)

> Warren Buffett · Charlie Munger · 段永平 · Li Lu — 四大师对抗式决策系统

## 🏰 简介

AI Berkshire 将四位价值投资大师的思维方式编译为可执行的决策系统，对你的持仓/标的做多视角对抗式研究。

- **段永平**：Right Business / Right People / Right Price + 本分测试
- **Buffett**：护城河识别 + DCF 三情景估值 + 交叉验证
- **Munger**：逆向死亡情景 + Lollapalooza 五维量化评分
- **Li Lu**：文明演进坐标 + 政策/地缘/供应链暗雷扫描

## 🚀 快速开始

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 功能模块

1. **核心指标卡** — 当前价 / 市值 / P/E / ROE
2. **四大师共识雷达图** — 5 轴 4 人轨迹
3. **Lollapalooza 评分条** — 五维量化 + 仓位档
4. **冲突点红黄灯** — 对抗式分析暴露真冲突
5. **三情景目标价** — 熊/基/牛 DCF 区间
6. **建仓建议卡** — 首仓/加仓/极限三档
7. **四大师详细分析** — 可展开的完整分析
